## Setup

Typography styles are included with all other styles in Indux CSS, or on a standalone basis.

<CodeGroup>
```html Indux CSS
<head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/indux/dist/indux.css" />
</head>
```
```html Standalone
<head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/indux/dist/indux.theme.css" />
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/indux/dist/indux.typography.css" />
</head>
```
</CodeGroup>

Styles make full use of theme variables, plus certain arbitrary property values for visual balance. Everything in the CSS files can be fully customized to suit your brand, or overridden in your HTML markup within classes.

## Theme

Default text elements use the following [theme](/styles/theme) variables:

| Variable | Purpose |
|----------|---------|
| `--color-content-stark` | High contrast text color |
| `--color-content-neutral` | Medium contrast text color |
| `--color-content-subtle` | Low contrast text color |
| `--color-text-inverse` | Text color inside form elements |
| `--spacing` | Padding and margin factor in various elements |
| `--radius` | Border radius for `<pre>` corners |
| `--transition` | Transition time for `<a>` states |

## Fonts

The global font and text color is set in the [theme](/styles/theme), with the default value being the user's system UI font or fallbacks. To apply a different font to individual text elements, you can update their [typography styles](/styles/text#setup).

`<pre>` and `<code>` elements use the user's system monospace font or fallbacks, set in the [reset](/styles/reset) styles if included.
