## Setup

Add an [Indux script](/getting-started/setup) to your project, or use the standalone URL queries plugin:

```html
<!-- Alpine -->
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

<!-- Indux URL Queries plugin -->
<script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.url.queries.js"></script>
```
