## Setup

Add an [Indux bundle](/getting-started/setup) to your project, or use the standalone tooltips plugin:

```html
<!-- Alpine -->
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

<!-- Indux Tooltips plugin -->
<script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.tooltips.js"></script>
```

---

Apply tooltips to any element with the `x-tooltip` attribute.

<Frame>
    <button x-tooltip="Hello world">Hover me</button>
</Frame>

```html
    <button x-tooltip="Hello world">Hover me</button>
```

---

## Anchoring
By default the tooltip appears centred below its element. The `x-tooltip` attribute name accepts parameters like `x-tooltip.bottom.left` that will anchor the tooltip to a different position.

```html
    <button x-tooltip.bottom.left="Hello world">Hover me</button>
```

Parameters are:
- `left`
- `right`
- `top` - centres above
- `top.left` - top left corner
- `top.right` - top right corner
- `bottom.left` - bottom left corner
- `bottom.right` - bottom right corner
- No parameter centres below

<Note>Tooltips will automatically flip sides to always appear onscreen.</Note>

## Rich Content
Tooltip values accept HTML.

<Frame>
    <button x-tooltip="Hello <b>world</b>">Hover me</button>
</Frame>

```html
    <button x-tooltip="Hello <b>world</b>">Hover me</button>
```

---

## Dynamic Content
Tooltips can be dynamic by using Alpine expressions.

<Frame>
    <div x-data="{ dynamicMessage: 'you son of a gun' }">
        <button x-tooltip="'Hello ' + dynamicMessage">Hover me</button>
    </div>
</Frame>

```html
<div x-data="{ dynamicMessage: 'you son of a gun' }">
    <button x-tooltip="'Hello ' + dynamicMessage">Hover me</button>
</div>
```

---

## Styles
Tooltips are styled by the `.tooltip` class in `elements.css`, including colors and offset (margin).


```html
<button x-tooltip="Basic tooltip">Hover me</button>
    <button x-tooltip="Press tooltip">Press me</button>
    <button x-tooltip.top="Top tooltip">Top</button>
    <button x-tooltip.top.right="Top right tooltip">Top right</button>
    <button x-tooltip.bottom="Bottom tooltip">Bottom</button>
    <button x-tooltip.left="Left tooltip">Left</button>
    <button x-tooltip.right="Right tooltip">Right</button>
    <button x-tooltip="'Dynamic: ' + dynamicMessage">Dynamic</button>
    <button x-tooltip="`Template literal: ${dynamicMessage}`">Template literal</button>
    ```
