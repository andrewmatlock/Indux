---
title: "Colors"
description: "Indux support light and dark modes out of the box."
---

Light mode uses the default color variables in `theme.css` or your custom `:root`. The same variables can be given dark mode overrides within any `.dark { ... }` declaration.

<CodeGroup>
    ```css theme.css```
    @theme {
        --color-page: 
    }
    ```
    ```css Custom :root```
    Hello
    ```
    ```css Nested overrides```
    Hello
    ```
</CodeGroup>

Indux's `theme.css` uses the Tailwind CSS variable naming convention with the `--color-` property prefix, making all colors available as utility classes.

## UI Toggles

<Frame>
    <button @click="setTheme('light')">Light</button>
    <button @click="setTheme('dark')">Dark</button>
    <button @click="setTheme('system')">System</button>
    <button @click="setTheme('toggle')" :icon="theme.current === 'light' ? 'ph:moon' : 'ph:sun'"></button>
</Frame>

```html
<div class="row gap-1">

## Current Theme

Display the current theme's title with `x-text="theme.current"` added to a text element.

```html
<p x-text="theme.current + ' theme'"></p>
```
