Bring individual Indux scripts and stylesheets into your project by CDN or CLI, or download them from [GitHub](https://github.com/andrewmatlock/Indux).

---

## Scripts
Indux scripts are plugins that extend [Alpine JS](https://alpinejs.dev) v3 or above.

There are three ways to bring them into your project.

<AccordionGroup>

  <Accordion title="All plugins, Alpine, and Tailwind included">

    `indux.alpine.tailwind.js` includes all plugins, with Alpine and Tailwind baked in. The script tag must include `defer` for Alpine to function.

    ```html
    <script defer src="https://cdn.jsdelivr.net/npm/indux/dist/indux.alpine.tailwind.js"></script>
    ```

    <Note>Tailwind code derives from the Play CDN script, with expansions to support custom utilities at runtime.</Note>
    

  </Accordion>

  <Accordion title="All plugins included">

    `indux.js` includes all plugins, and requires Alpine JS to run. Alpine's script tag must include `defer` to function.

    ```html
    <!-- Alpine -->
    <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

    <!-- Indux (all plugins) -->
    <script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.js"></script>
    ```
  </Accordion>

  <Accordion title="Individual plugins">


    For projects requiring only certain components, load them individually to reduce payload.

    ```html
    <!-- Alpine -->
    <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

    <!-- Indux (individual plugins) -->
    <script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.components.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.icons.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.themes.js"></script>
    ```

    CDN links for each can be found in their plugin articles.

  </Accordion>

</AccordionGroup>


<Note>All scripts can be placed anywhere in the `<head>` or `<body>`, in any order.<br/>
The bundled scripts (or the standalone `indux.components.js` plugin) will manage page routing, and scripts only need to be applied once to `index.html`.</Note>



<Tabs>

    <Tab title="CDN">
        ```html
        <!-- Alpine -->
        <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>

        <!-- All Indux plugins -->
        <script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.js"></script>

        <!-- OR individual Indux plugins by name (see plugin articles) -->
        <script src="https://cdn.jsdelivr.net/npm/indux/dist/indux.components.js"></script>
        ```
    </Tab>

    <Tab title="CLI">
        For the bundled script containing all plugins:
        ```node
        npx @indux/all
        ```

        OR individual plugins by name (see plugin articles):
        ```node
        npx @indux/components
        ```

        These commands will install the plugins in a `/scripts` folder, or update existing plugins (with matching names, e.g. `indux.components.js`) wherever they live in your project.
    </Tab>

</Tabs>

## Stylesheets
Add the desired Indux CSS stylesheets to the `<head>` of your HTML file.

<Tabs>

    <Tab title="CDN">
        ```html
        <head>
            <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/indux/dist/indux.normalizer.css">
            <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/indux/dist/indux.base.css">

            <!-- Your additive CSS in one or more files -->
            <!-- Include Indux's CSS variables required by `indux.base.css` -->
            <link rel="stylesheet" href="styles/main.css">
            <link rel="stylesheet" href="styles/theme.css">

        </head>
        ```

        <Note>Get Indux's CSS variables from [Github](https://github.com/indux/indux/blob/main/styles/indux.theme.css)</Note>

    </Tab>

    <Tab title="CLI">
        Install or update local Indux scripts, and reference them in your `<head>`. This CLI command will install the sheets in a `/styles` folder, or update existing sheets with matching names wherever they live in your project.
        ```node
        npm install @indux/styles
        ```
    </Tab>

</Tabs>
