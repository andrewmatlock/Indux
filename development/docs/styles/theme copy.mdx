### Overview

Indux provides a comprehensive set of base styles and utility classes that integrate HTML elements into your global design system.

<Note>Many base styles and utility classes reference the [project theme](/styles/theme) for visual consistency. Take care when modifying or removing the theme's CSS variables.</Note>

---

### Base Styles

The following HTML tags (and their available utility classes) are styled to support a consistent UI with minimalist HTML markup.

| Category | Elements |
|----------|----------|
| **Text** | `h1`-`h6`, `p`, `a`, `small`, `strong`, `em`, `code`, `pre`, `blockquote`, `figcaption`, `span`, `b`, `ins`, `del`, `abbr`, `kbd`, `samp`, `mark`, `sub`, `sup`, `q`, `cite`, `address`, `ul`, `ol`, `li`, `dl`, `dt`, `dd` |
| **Forms** | `button`, `input`, `select`, `textarea`, `button`, `label`, `optgroup`, `progress` |
| **Layout** | `body`, `header`, `footer`, `main`, `section`, `aside`, `nav`, `hr`, `figure` |
| **Menus**| `details`, `summary`, `menu`, `dialog`, `header`, `main`, `footer` |
| **Navigation** | `header`, `footer`, `aside` |
| **Tables** | `table`, `thead`, `tbody`, `tfoot`, `tr`, `th`, `td` |

### Utility Classes

Extending Tailwind CSS, Indux provides a set of useful utility classes that consolidate common styles.

#### Layout

| Class | Description | Tailwind Equivalent |
|--------------|-------------|-------------------|
| `content` | Contains content to middle of page | `w-(--spacing-content-width) max-w-full mx-auto` |
| `row` | Horizontal flex container | `flex flex-row` |
| `row-wrap` | Horizontally wrapping flex container | `flex flex-row flex-wrap` |
| `col` | Vertical flex container | `flex flex-col` |
| `col-wrap` | Vertically wrapping flex container | `flex flex-col flex-wrap` |
| `center` | Centers content | `justify-center items-center` |

#### Form Elements

| Class | Type | Description |
|--------------|-----------|---------------------|
| `primary` | Color | Primary or contrast color |
| `accent` | Color | Accent or brand color |
| `negative` | Color | Negative or warning color |
| `ghost` | Appearance | Transparent background until hovered or pressed |
| `hug` | Appearance | Sizes to its content |
| `transparent` | Appearance | Transparent background in all states |
| `outlined` | Appearance | Bordered style |
| `lg` | Size | Larger variant |
| `sm` | Size | Smaller variant |

#### Miscellaneous

| Class | Description |
|--------------|--------------------------------|
| `no-focus` | Removes an element's focus outline |
| `no-scrollbar` | Hides an element's scrollbar |
| `overlay-dark` | For banner elements with a dark overlay and light text inside |
| `overlay-light` | For banner elements with a light overlay and dark text inside |
| `prose` | Makes elements with long-form, child text content (e.g. articles) more readable |
| `trailing` | Colors and pushes a trailing text or icon element to the right |
