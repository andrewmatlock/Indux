Test Instructions

  • Hover/Focus/Active: Interact with buttons to see state changes
  • Form States: Fill out form fields to see validation states
  • Group/Peer: Hover over group containers and interact with peer elements
  • Data/Aria: Elements with data-state and aria attributes should show different styling
  • Media Queries: Resize window, toggle dark mode, and check print preview
  • Responsive: Resize browser window to see breakpoint changes
  • Opacity: Background colors should have transparency applied
  • RTL: Add dir="rtl" to body to test RTL variants
